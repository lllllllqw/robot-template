# Robot Template
## Description

Template for [Create WeCom Robot App](https://github.com/lllllllqw/create-wecom-robot-app)

## Installation

```bash
$ npx create-wecom-robot-app my-robot
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## a
The template is based [Nest](https://github.com/nestjs/nest)
